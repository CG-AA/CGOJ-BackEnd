Handles the user registration process.

This function sets up a POST route at "/register" to handle user registration requests. It includes rate limiting, user data validation, password hashing, and JWT token generation.

Parameters
appReference to the CROW application object, used to define the route.
settingsReference to a JSON object containing application settings.
IPString representing the client's IP address, used for logging.
apiUnique pointer to the APIs object, facilitating database interactions.
Attention
Email ownership verification should be implemented to prevent unauthorized registrations.
